# Система обліку вітроенергетичних установок (WES Registry)

Простий та ефективний вебдодаток для моніторингу та реєстрації об'єктів вітрової енергетики. Дозволяє керувати списком турбін, розраховувати їх сумарну потужність та візуалізувати локації на карті.



## Основні функції
* **Додавання турбін:** Форма з валідацією технічних характеристик (потужність, висота, діаметр ротора).
* **Автоматичні розрахунки:** Сервер самостійно обчислює загальну потужність об'єкта.
* **Інтерактивна карта:** Миттєвий перехід до Google Maps за вказаними координатами.
* **Збереження даних:** Всі дані записуються у локальну базу `turbines.json`.
* **Адаптивний дизайн:** Зручний інтерфейс для ПК та мобільних пристроїв.



## Технологічний стек
* **Backend:** Node.js, Express.js
* **Frontend:** JavaScript (ES6+), HTML5, CSS3 (Glassmorphism design)
* **Database:** JSON-based storage



## Структура проєкту
* `server.js` — Логіка сервера та API маршрути.
* `public/` — Клієнтська частина (HTML, CSS, JS).
* `turbines.json` — Файл бази даних.



## Як запустити проєкт локально

1.  **Встановіть Node.js**, якщо він ще не встановлений.
2.  **Завантажте проєкт** у робочу папку.
3.  Відкрийте термінал у цій папці та встановіть залежності (якщо використовується Express):
    ```bash
    npm install
    ```
4.  Запустіть сервер:
    ```bash
    node server.js
    ```
5.  Відкрийте браузер та перейдіть за адресою:
    ```
    http://localhost:3000
    ```



## Запуск на мобільних пристроях
Для перегляду з телефону введіть свою локальну IPv4-адресу в мережі Wi-Fi (наприклад, `http://192.168.0.155:3000`).
